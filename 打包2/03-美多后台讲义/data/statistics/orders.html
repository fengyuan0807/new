日下单用户量统计

本小节目标：

  • 理解网站日下单用户数统计API接口设计与实现

1. 接口设计

请求方式： GET /meiduo_admin/statistical/day_orders/

请求参数： 通过请求头传递jwt token数据。

返回数据： JSON

{
        "count": "下单用户量",
        "date": "日期"
}
返回值 类型 是否必须 说明
count int 是 下单用户量
date date 是 日期

2. API实现

1）在meiduo_admin/views目录下的statistical.py文件中实现统计日下单用户数的接口。

from rest_framework.views import APIView
from rest_framework.response import Response
from rest_framework.permissions import IsAdminUser
from django.utils import timezone

from users.models import User

# GET /meiduo_admin/statistical/day_orders/
class UserDayOrdersView(APIView):
    permission_classes = [IsAdminUser]

    def get(self, request):
        """
        获取日下单用户数量：
        1. 获取日下单用户数量
        2. 返回应答
        """
        # 1. 获取日下单用户数量
        now_date = timezone.now().replace(hour=0, minute=0, second=0, microsecond=0)
        count = User.objects.filter(orders__create_time__gte=now_date).distinct().count()

        # 2. 返回应答
        response_data = {
            'date': now_date.date(),
            'count': count
        }

        return Response(response_data)

2）在meiduo_admin/urls.py文件中添加配置项

from django.conf.urls import url
from meiduo_admin.views import statistical

urlpatterns = [
    # ...
    url(r'^statistical/day_orders/$', statistical.UserDayOrdersView.as_view()),
]

总结：

  • 核心代码：User.objects.filter(orders__create_time__gte=now_date).distinct().count()
