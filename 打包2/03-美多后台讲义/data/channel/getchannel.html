获取频道表数据

本小节目标：

  • 理解频道数据获取API接口设计与实现

频道获取

1. 接口设计

请求方式： GET /meiduo_admin/goods/channels/?page=<页码>&page_size=<页容量>

请求参数： 通过请求头传递jwt token数据。

返回数据： JSON

    {
        "counts": "频道总数量",
        "lists": [
            {
                "id": "频道ID",
                "category": "一级分类名称",
                "category_id": "一级分类ID",
                "group": "频道组名称",
                "group_id": "频道组ID",
                "url": "频道页面链接地址",
                "sequence": "频道展示顺序"
            },
            ...
        ],
        "page": "页码",
        "pages": "总页数",
        "pagesize": "页容量"
    }
返回值 类型 是否必须 说明
count int 是 频道总量
lists 数组 是 频道数据
page int 是 页码
pages int 是 总页数
pagesize int 是 页容量

2. API实现

1）在meiduo_admin/views目录下新建channels.py文件，并实现API接口。

from rest_framework.permissions import IsAdminUser
from rest_framework.viewsets import ModelViewSet

from goods.models import GoodsChannel
from meiduo_admin.serializers.channels import ChannelSerializer


# GET /meiduo_admin/goods/channels/?page=<页码>&page_size=<页容量>
class ChannelViewSet(ModelViewSet):
    """频道管理视图集"""
    permission_classes = [IsAdminUser]
    # 指定视图所使用的查询集
    queryset = GoodsChannel.objects.all()

    # 指定序列化器类
    serializer_class = ChannelSerializer

2）在meiduo_admin/serializers目录下新建channels.py文件，并定义序列化器类。

from rest_framework import serializers

from goods.models import GoodsChannel


class ChannelSerializer(serializers.ModelSerializer):
    """频道序列化器类"""
    # 关联对象嵌套序列化
    category = serializers.StringRelatedField(label='一级分类名称')
    group = serializers.StringRelatedField(label='频道组名称')

    category_id = serializers.IntegerField(label='一级分类ID')
    group_id = serializers.IntegerField(label='频道组ID')

    class Meta:
        model = GoodsChannel
        exclude = ('create_time', 'update_time')

3）在meiduo_admin/urls.py文件中进行路由配置。

# 频道管理
from rest_framework.routers import DefaultRouter
router = DefaultRouter()
router.register('goods/channels', channels.ChannelViewSet, base_name='channels')
urlpatterns += router.urls

总结：

  • 频道数据获取API：
    • channel.category：和频道关联的一级分类
    • channel.group：和频道关联的频道组
    • StringRelatedField：将关联对象嵌套序列化为关联对象模型类_str_方法的返回值
