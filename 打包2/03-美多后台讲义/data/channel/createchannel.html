新增频道表数据

本小节目标：

  • 理解频道组数据获取API接口的设计与实现
  • 理解一级分类数据获取API接口的设计与实现
  • 理解频道数据新增API接口的设计与实现

频道新增

新增频道之前需要先获取所有一级分类数据和频道组的数据。

1. 获取频道组数据

1.1 接口设计

请求方式： GET /meiduo_admin/goods/channel_types/

请求参数： 通过请求头传递jwt token数据。

返回数据： JSON

    [
        {
            "id": "频道组id",
            "name": "频道组名称"
        },
        ...
    ]
返回值 类型 是否必须 说明
id int 是 频道组id
name int 是 频道组名称

1.2 API实现

1）在meiduo_admin/views/channels.py文件中实现API接口。

from goods.models import GoodsChannelGroup
from meiduo_admin.serializers.channels import ChannelGroupSerializer

# GET /meiduo_admin/goods/channel_types/
class ChannelTypesView(ListAPIView):
    """频道组视图"""
    permission_classes = [IsAdminUser]
    # 指定视图所使用的查询集
    queryset = GoodsChannelGroup.objects.all()

    # 指定序列化器类
    serializer_class = ChannelGroupSerializer

    # 注：关闭分页
    pagination_class = None

2）在meiduo_admin/serializers/channels.py文件中定义序列化器类ChannelGroupSerializer。

from goods.models import GoodsChannelGroup

class ChannelGroupSerializer(serializers.ModelSerializer):
    """频道组序列化器类"""
    class Meta:
        model = GoodsChannelGroup
        fields = ('id', 'name')

3）在meiduo_admin/urls.py文件中进行路由配置。

urlpatterns = [
    # ...
    # 频道管理
    url(r'^goods/channel_types/$', channels.ChannelTypesView.as_view()),
]

2. 获取一级分类数据

2.1 接口设计

请求方式： GET /meiduo_admin/goods/categories/

请求参数： 通过请求头传递jwt token数据。

返回数据： JSON

    [
        {
            "id": "分类id",
            "name": "分类名称"
        },
        ...
    ]
返回值 类型 是否必须 说明
id int 是 分类id
name int 是 分类名称

2.2 API实现

1）在meiduo_admin/views/channels.py文件中实现API接口。

from rest_framework.generics import ListAPIView

from goods.models import GoodsCategory
from meiduo_admin.serializers.channels import ChannelCategorySerializer


# GET  `/meiduo_admin/goods/categories/`
class ChannelCategoriesView(ListAPIView):
    """频道对应一级分类视图"""
    permission_classes = [IsAdminUser]
    # 指定视图所使用的查询集
    queryset = GoodsCategory.objects.filter(parent=None)

    # 指定序列化器类
    serializer_class = ChannelCategorySerializer

    # 注：关闭分页
    pagination_class = None

2）在meiduo_admin/serializers下的channels.py文件定义序列化器类ChannelCategorySerializer。

from rest_framework import serializers
from goods.models import GoodsCategory


class ChannelCategorySerializer(serializers.ModelSerializer):
    """商品分类序列化器类"""
    class Meta:
        model = GoodsCategory
        fields = ('id', 'name')

3）在meiduo_admin/urls.py文件中进行路由配置。

urlpatterns = [
    # ...
    # 频道管理
    url(r'^goods/categories/$', channels.ChannelCategoriesView.as_view()),
]

3. 保存新增频道数据

3.1 接口设计

请求方式： POST /meiduo_admin/goods/channels/

请求参数： 通过请求头传递jwt token数据。

    {
        "category_id": "一级分类ID",
        "group_id": "频道组ID",
        "url": "频道页面链接地址",
        "sequence": "频道展示顺序"
    }
返回值 类型 是否必须 说明
category_id int 是 一级分类ID
group_id int 是 频道组ID
url str 是 频道页面地址
sequence int 是 频道展示顺序

返回数据： JSON

    {
        "id": "频道ID",
        "category": "一级分类名称",
        "category_id": "一级分类ID",
        "group": "频道组名称",
        "group_id": "频道组ID",
        "url": "频道页面链接地址",
        "sequence": "频道展示顺序"
    }
返回值 类型 是否必须 说明
id int 是 频道ID
category_id int 是 一级分类ID
group_id int 是 频道组ID
category str 是 一级分类名称
group str 是 频道组名称
url str 是 频道页面地址
sequence int 是 频道展示顺序

3.2 API实现

1）在meiduo_admin/serializers/channels.py文件中修改ChannelSerializer序列化器类，补充验证方法。

class ChannelSerializer(serializers.ModelSerializer):
    """频道序列化器类"""
    # 关联对象嵌套序列化
    category = serializers.StringRelatedField(label='一级分类名称')
    category_id = serializers.IntegerField(label='一级分类ID')

    # 关联对象嵌套序列化
    group = serializers.StringRelatedField(label='频道组名称')
    group_id = serializers.IntegerField(label='频道组ID')

    class Meta:
        model = GoodsChannel
        exclude = ('create_time', 'update_time')

    def validate_category_id(self, value):
        # 一级分类是否存在
        try:
            GoodsCategory.objects.get(id=value, parent=None)
        except GoodsCategory.DoesNotExist:
            raise serializers.ValidationError('一级分类不存在')

        return value

    def validate_group_id(self, value):
        # 频道组是否存在
        try:
            GoodsChannelGroup.objects.get(id=value)
        except GoodsChannelGroup.DoesNotExist:
            raise serializers.ValidationError('频道组不存在')

        return value

注意：post /meiduo_admin/goods/channels/时，会调用ChannelViewSet视图集中的create方法，已经满足这里的需求，直接使用即可。
