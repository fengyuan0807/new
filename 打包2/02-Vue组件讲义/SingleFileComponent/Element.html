Element-ui插件

Element-Ul是饿了么前端团队推出的一款基于Vue.js 2.0 的桌面端UI框架，内部封装了很多单文件组件，根据需要，我们可以直接将这些单文件组件引入到自己的项目中。

Element官网文档： http://element-cn.eleme.io/#/zh-CN/component/progress

1. 基本使用

1）在main.js中将emelent-ui引入到项目中。

import Vue from 'vue'
import App from './App.vue'

// 导入ElementUI插件
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

// 在vue中使用Element-UI插件
Vue.use(ElementUI)

// 导入定义的路由规则
import router from './router/router.js'


new Vue({
    el: "#app",
    // 让Vue使用路由规则
    router: router,
    render: function (create) {
        return create(App);
    }
});

2）在meiduo/components目录下新建Button.vue组件，编写内容：

<template>
    <div>
        <el-row>
          <el-button>默认按钮</el-button>
          <el-button type="primary">主要按钮</el-button>
          <el-button type="success">成功按钮</el-button>
          <el-button type="info">信息按钮</el-button>
          <el-button type="warning">警告按钮</el-button>
          <el-button type="danger">危险按钮</el-button>
        </el-row>
    </div>
</template>

<script>
    export default {

    }
</script>

<style scoped>

</style>

注：需要使用Element封装的单文件组件，直接到官网文档上找到所需的效果，查看源码并复制引入到项目中即可。

3）在router.js文件中添加路由规则。

import Vue from 'vue'
// 导入路由插件
import Router from 'vue-router'

import Child1 from '../components/Child1.vue'
import Child2 from '../components/Child2.vue'
// 此处为新内容
import Button from '../components/Button.vue'


// 1. 在vue中使用路由插件
Vue.use(Router);

// 2. 定义路由规则
export default new Router({
    routes: [
        {
            path: '/',
            component: Child1
        },
        {
            path: '/child2',
            component: Child2
        },
        // 此外为新内容
        {
            path: '/button',
            component: Button
        }
    ]
})
